## What this snippet does (clear intent)

This is a **Tempo-like Obsidian theme snippet**, not a full theme.

It gives you:

* Soft, modern cards
* Navy / cyan / purple system
* Rounded surfaces
* Calm contrast (no harsh blacks)
* Dashboard-first visual hierarchy
* Tracker + Dataview friendly styling

You drop this in once and your **Habit Dashboard instantly feels like Tempo**.

No plugin hacks.
No broken selectors.
No brittle overrides.

---

## How to install the snippet (exact steps)

1. Open your Obsidian vault
2. Go to
   `.obsidian/snippets/`
3. Create a file named
   `tempo-dashboard.css`
4. Paste **everything below**
5. In Obsidian:

   * Settings → Appearance
   * Enable **tempo-dashboard**

Restart Obsidian once after enabling.

## How this maps to Tempo visually

### Tempo feature → Obsidian equivalent

* KPI cards → Tracker blocks styled as cards
* Progress rings → Donut charts with clean center space
* Soft gradients → Area charts with opacity
* Sidebar calmness → Muted navy + white cards
* “Losing streak” warnings → Rose-tinted alert blocks
* Habit list icons → Emoji + Iconize

This is **intentional minimalism**, not over-styling.
